About Project

Graphical password authentication is a security mechanism that replaces traditional alphanumeric passwords with graphical elements. Instead of typing a sequence of characters, users select images from from user device system to create a password. The idea behind graphical passwords is to leverage human visual memory and recognition abilities, making it easier for users to remember their passwords while maintaining security.
